# 连接 CloudBase MCP

通过 Model Context Protocol (MCP) 连接后，你可以在 AI 对话中直接操作 CloudBase 服务，无需在 IDE 和控制台之间切换。

## 前置条件

在开始配置之前，请确保满足以下条件：

<details>
<summary><strong>已准备好Node.js 环境和云开发环境</strong></summary>

**Node.js 环境：** 确保已安装 Node.js v18.15.0 或更高版本：

```bash
node --version
```

如果未安装，请从 [Node.js 官网](https://nodejs.org/en/download) 下载安装。

**云开发环境：** 请参考文档 [开通云开发环境](https://docs.cloudbase.net/quick-start/create-env)。新用户可以免费开通体验。

</details>

## 手动确认工具调用

大多数 MCP 客户端（如 Cursor）会在执行每个工具调用前要求你手动确认。我们建议你始终启用此设置，并在执行前仔细查看工具调用的详细信息。

## 开始使用

配置完成后，在 AI 对话中输入：

```
登录云开发
```

AI 会自动引导你完成登录和环境选择。然后你可以开始使用自然语言描述需求，让 AI 帮你开发应用。

## 使用示例

**完整流程示例：**

```
帮我创建一个待办应用并部署
```

AI 会自动完成：生成代码、创建数据库、部署云函数、配置静态托管，并返回访问链接。

**分步操作示例：**

- "创建用户表" - AI 自动创建数据库集合
- "部署这个函数" - AI 自动上传并配置云函数  
- "部署前端到 CDN" - AI 自动上传文件并配置托管

**详细需求示例：**

```
帮我创建一个待办事项应用，包含：
- 添加、删除、编辑待办事项
- 数据存储到云数据库
- 支持标记完成状态
- 部署到云端
```
