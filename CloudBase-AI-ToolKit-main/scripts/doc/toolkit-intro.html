02

智能登录认证

AI 自动识别项目类型，Web 和小程序使用不同的认证方式。你不需要了解技术细节，AI 帮你搞定一切。

✓ Web 项目：自动使用 CloudBase Web SDK 内置认证
✓ 小程序项目：自然免登录，在云函数中获取 OPENID
✓ 减少配置错误，开箱即用
// AI 自动选择认证方式
if (projectType === 'web') {
// 使用 Web SDK 内置认证
const auth = app.auth();
auth.signInAnonymously();
} else if (projectType === 'miniprogram') {
// 小程序自然免登录
const openid = cloud.getWXContext().OPENID;
}
// AI 自动识别，无需手动配置

技术突破：Skill 体系

从规则到技能，让 AI 更专业

为什么 Skill 体系如此重要？

在 1.x 版本中，我们使用规则（Rules）来指导 AI。但规则是分散的、缺乏结构的。2.0 版本引入 Skill 体系，将规则重构为结构化的技能模块，让 AI 能够像专业开发者一样思考和工作。

1.x 规则体系
  • 分散的规则文件
  • 缺乏结构化组织
  • 难以组合和复用
  • AI 理解困难
  • 代码质量不稳定
2.0 Skill 体系
  • 结构化的技能模块
  • 完整的知识体系
  • 支持技能组合和复用
  • AI 理解更准确
  • 代码质量显著提升
